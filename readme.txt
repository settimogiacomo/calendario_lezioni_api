Lato backend di calendario lezioni
API con node.js

per attivare: 
node index.js

per installare:
npm install express
npm install cors: for enabling CORS
npm install body-parser: Parse incoming request bodies in a middleware before your handlers, available under the req.body property.
npm install mysql

per uscire:
control + C
oppure .exit

per aggiornare: 
spegnere e riaccendere il server
